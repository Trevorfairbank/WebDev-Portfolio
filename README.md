## About
My Name is Trevor Fairbank and this is my portfolio! The image on the homepage is a photo I captured in Lake Tahoe, my home town. I am a photographer and am starting my career path in the tech industry to become a web developer. You can view the applications I have worked on within the projects section. On hover you will be able to click on the github icon to see the code, and the globe icon to see the live application. I created these applications during my time at a Bootcamp through UC San Diego Extension.
